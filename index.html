About FINDerr

The short version.

FINDerr is a shared board for the class to log and find study materials — notes, past questions, tutorial sheets, whatever's useful. Entries here are visible to everyone who opens this page, so treat it like a shared logbook: log what you've got, hoist what helps you, and keep it accurate for the next person.

Files themselves live wherever you already store them (Drive, Dropbox, etc.) — FINDerr just keeps the map to find them.

One-time setup

This copy of FINDerr needs a database to save entries for real, from any browser, forever. It takes about 5 minutes and it's free.

1. Go to supabase.com → sign up (free) → New project.

2. Once it's created, open the SQL Editor (left sidebar) and run this:

create table materials (
  id text primary key,
  title text, course text, type text,
  description text, link text, tags text,
  uploader text, votes integer default 0,
  created_at bigint
);
alter table materials enable row level security;
create policy "public read" on materials
  for select using (true);
create policy "public insert" on materials
  for insert with check (true);
create policy "public update" on materials
  for update using (true);

3. Go to Project Settings → API. Copy the Project URL and the anon public key.

4. Open this HTML file in a text editor, find SUPABASE_URL and SUPABASE_ANON_KEY near the top of the <script> tag, paste your two values in, save, and reload this page.

5. To share it with the class, host the file for free on Netlify Drop (just drag the file in) or GitHub Pages, and send the link.
